# mikroOmega

Malá simulace akvárka

Kup ryby, krm je, koukej jak plavou a nech je vydělávat coiny. Když na ně kašleš, hladoví a zmenšují se.
Prostě mini simulace akvária v Javě.

## Co v tom je

- menu, akvárium a obchod
- ryby s vlastním jménem, barvou, hladem a velikostí
- pasivní coiny podle velikosti ryb
- krmení z obchodu
- jednoduchá animace plavání

## Spuštění

V kořeni projektu:

```bash
javac -d out src/*.java
java -cp out Main
```

Nebo to otevři v IntelliJ IDEA jako obyčejný Java projekt a spusť `Main`.

## Poznámka

Projekt nemá žádné externí knihovny. Stačí JDK a chuť vlastnit pár digitálních ryb.
